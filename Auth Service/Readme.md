# Auth Service 


## Functions :
    1. Register
    2. Login
    3. Verify Token
    4. Manage Profile

## Steps to run :
### Step 1: Create a virtual environment
```shell
python -m venv venv
```
### Step 2: Activate the virtual environment
```shell
venv/Scripts/activate
```

### Step 3: Install all requirements
```shell
pip install -r requirements.txt
```
    
    
### Step 4: Run the source file
```shell
python main.py
```
